Charon Manifest Installer v1.0.2

Portable console manifest downloader.

What it does:
- Checks for updates on startup.
- Finds Steam automatically.
- Reads Lua files from Steam\config\stplug-in.
- Downloads manifests into Steam\depotcache.
- Shows whether the manifest came from ManifestVault, External Database, or ManifestHub.
- Press O or 0 after a run to open the depotcache folder.
- Press Z after a run to preview and create an AppID ZIP in Documents.

Source order:
- Mode 1: ManifestVault -> External Database
- Mode 2: ManifestVault -> External Database -> ManifestHub

No installer required. Run CharonManifestInstaller.exe directly.
